GET copy_get_params
Создание сессии копирования, получение параметров копирования

Входные параметры:
нет

Выходные параметры:
id_session - ид сессии
params - массив параметров

Пример:

{
    "id_session": "82f90ad7-45fc-4f06-a902-e5f62934ba62",
    "params": {
        "copy-one-side": 10.0,
        "copy-two-side": 17.0
    }
}

POST start_simple_copy
Простое копирование (сканирование + печать документов при одностороннем копировании или сканирование х2 запроса + печать документа на второй итерации)

Входные параметры:
id_session - ид сессии
count_copy - кол-во копий (целое число). Необязательный параметр, по умолчанию 1.
type_print - тип копирования (two - двустороннее копирование, one - одностороннее). Необязательный параметр, по умолчанию one.
number_scan - номер итерации копирования (целое число). Необязательный параметр, по умолчанию 1.
#number_scan применяется при двустороннем копировании. Т.е. отсканировали первую сторону - number_scan = 1, отсканировали вторую сторону - number_scan = 2.
 
Выходные параметры:
id_session - ид сессии
res - результат копирования (при односторонней печати или 2 итерации сканирования)

----------------------------------------------
POST start_identification_copy
Сканирование удостоверений личности (2 А5 на 1 А4)

Входные параметры:
id_session - ид сессии

Выходные параметры:
id_session - ид сессии
res - массив с именами сканов (.jpg) в папке пользователя (для предпросмотра)

----------------------------------------------
POST print_identification_copy
Печать сканов удостоверения личности (объединяет все файлы в папке пользователя и печатает)(2 А5 на 1 А4)

Входные параметры:
id_session - ид сессии
count_copy - кол-во копий (целое число). Необязательный параметр, по умолчанию 1.

Выходные параметры:
res - результат печати удостоверения личности

----------------------------------------------
DELETE delete_copy
Удаление файла из папки пользователя 

Входные параметры:
id_session - ид сессии
image_name - имя файла (например, img.jpg)

Выходные параметры:
id_session - ид сессии
res - массив оставшихся сканов в папке
