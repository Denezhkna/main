POST start_scan
Запуск сканирования, сохранение в папку пользователя, создание сессии (если ранее не создавалась)

Входные параметры:
id_session - ид сессии (если json - none, создается новая сессия)

Выходные параметры:
id_session - ид сессии,
res - массив с именами изображений

----------------------------------------------
GET scan_get_params
Создание сессии (папки пользователя) и получение параметров печати

Входные параметры:
нет

Выходные параметры:
id_session - ид сессии,
"params": [  - массив параметров
        {
            "operation": {
                "name": "scan" - наименование действия
            },
            "price": стоимость скана (1 шт.)
        }

----------------------------------------------
POST rotate_image
Поворот изображения на заданный угол

Входные параметры:
id_session - ид сессии,
image_name - имя изображения (с расширением)
angle - угол/градус поворота изображения ЦЕЛОЕ ЧИСЛО 

Выходные параметры:
res - имя нового изображения

----------------------------------------------
DELETE delete_scan
Удаление скана по наименованию 

Входные параметры:
id_session - ид сессии,
image_name - имя изображения (с расширением, 1 шт)


Выходные параметры:
id_session - ид сессии
res - массив с именами изображений в папке пользователя

----------------------------------------------
POST save_or_send_scan
Сохрание или отправка файлов (usb или email)

Входные параметры:
id_session - ид сессии
format_save - формат сохранения. Допустимые значения ".pdf" и ".jpg"
type_file - тип файла. Допустимые значения "one" - одностраничный, "multi" - многостраничный
save - вложенный json с параметрами сохранения
type - тип сохранения. Допустимые значения "email" и "usb"
path - путь сохранения. Если email, то адрес получателя. Если usb, то пропустить

Пример 1:
{
    "id_session":"cdf5cf8c-87cd-4891-ac14-991991c9c1ab",
    "format_save":".jpg",
    "type_file":"one",
    "save":{
        "type":"usb"
    }
}

Пример 2:
{
    "id_session":"cdf5cf8c-87cd-4891-ac14-991991c9c1ab",
    "format_save":".pdf",
    "type_file":"multi",
    "save":{
        "type":"email",
        "path":"denezhkina@baltoftech.ru"
    }
}

Выходные параметры:
res - результат отправки/сохранения (текст)
